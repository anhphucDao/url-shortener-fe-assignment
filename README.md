# URL Shortener — DevCamp 2026

## Overview
**DevCamp URL Shortener** is a modern frontend web application designed to simplify URL management. This project enables users to input lengthy web addresses and instantly generate compact, shareable short links. Built as part of the DevCamp 2026 assignment, it features an intuitive and responsive user interface, handling form submissions seamlessly and displaying the result in a clean, user-friendly modal.
